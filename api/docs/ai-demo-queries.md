# AI 검색 데모 질의셋 (OI5)

이 문서는 라우터(`router_node`)와 세 경로(A/B/C)의 기대 동작을 명세하는 **단일출처**다.
라우터 프롬프트 예시·단위 테스트 케이스·4.8 검증 매트릭스가 이 표를 참조한다.

- **경로 A (구조형, FR13/FR14)**: 가격·차종·연식·색상·지역·주행거리·연료 등 **명시적 조건** → Text-to-SQL.
- **경로 B (질적·의미형, FR15)**: 정확한 조건 대신 **용도·느낌·추천** → 문서 RAG(임베딩 유사도).
- **경로 C (매물 무관, FR16)**: 중고차 검색과 무관한 잡담·상식 → **정중한 거절 + 검색 유도**(listings 빈 목록).

회색지대(차 얘기인데 조건이 흐릿)는 **B로 보내 추천**한다(빈손보다 추천이 낫다). 정교한 되묻기(clarify) 멀티턴은 4.6 이후.

응답 계약은 모든 경로 공통으로 `{answer, listings[]}`다. 0건이어도 빈손이 아니라 조건 완화/재질문 안내를 준다(FR17).

---

## ① 구조형 → 경로 A

| 질의 | 기대 분류 | 기대 결과 요지 |
|---|---|---|
| "3천만원 이하 흰색 SUV" | A | `price <= 30000000 AND color='흰색' AND body_type='SUV' AND status='on_sale'` 매물 카드. |
| "2020년 이후 제네시스" | A | `manufacturer='제네시스' AND year >= 2020` 매물. |
| "10만km 미만 디젤" | A | `mileage < 100000 AND fuel='디젤'` 매물. |
| "서울 경차 보여줘" | A | `region='서울' AND body_type='경차'` 매물. |

## ② 질적·의미형 → 경로 B

| 질의 | 기대 분류 | 기대 결과 요지 |
|---|---|---|
| "패밀리카로 무난한 거" | B | 임베딩 유사도로 가족용에 어울리는 on_sale 매물 + 근거 가이드 1건. |
| "초보운전자에게 좋은 차" | B | 의미 유사 매물 추천(정확 조건 없음). |
| "연비 좋은 전기차 추천" | B | 의미형 추천(전기차 키워드 있어도 "추천" 의도라 B). |
| "가성비 좋은 차 없을까?" | B | 느낌·용도 기반 추천. |

## ③ 회색지대 (애매 — 기대 처리 명시: B 우선)

| 질의 | 기대 분류 | 기대 결과 요지 |
|---|---|---|
| "출퇴근용 적당한 차" | B | 용도만 있고 조건 흐릿 → B로 추천(되묻기 X). |
| "괜찮은 SUV 있어?" | B | 차종 1개뿐 + "괜찮은"이라는 주관 → 조건이 흐릿하므로 B. (LLM이 A로 봐도 카드를 주면 데모상 허용.) |
| "너무 비싸지 않은 중형차" | B 또는 A | "비싸지 않은"이 모호 → B 권장. LLM이 A로 판정해 가격 임계로 풀어도 무방. |

> 회색지대는 LLM 판정이 A/B로 갈릴 수 있다. 둘 중 어디로 가도 **매물 카드 또는 추천을 돌려주면** 데모 목적상 합격(빈손·거절만 아니면 됨).

## ④ 매물 무관 → 경로 C (가드 거절)

| 질의 | 기대 분류 | 기대 결과 요지 |
|---|---|---|
| "오늘 날씨 어때?" | C | "저는 중고차 매물 검색을 도와드리는 어시스턴트예요…" + `listings: []`. |
| "파이썬 코드 짜줘" | C | 정중한 거절 + 검색 유도. 상식/코드 Q&A 미제공(FR16). |
| "안녕" | C | 인사도 거절 문구로 검색 유도. |
| "1+1은 뭐야?" | C | 상식 질의 거절. |

## 차형(body_type) 용어 매핑 (경로 A SQL 생성 참고)

자연어 차형 표현을 DB `body_type` 허용값으로 매핑한다(단일출처: `sql_rag_node` 프롬프트).

| 자연어 | DB 매핑 |
|---|---|
| 세단 | `body_type IN ('준중형차','중형차','대형차')` |
| SUV | `body_type = 'SUV'` |
| 경차 | `body_type = '경차'` |
| 데모에 없는 차형(해치백·쿠페 등) | 무리한 매핑 금지 — 가격·기타 조건만 적용 |

> `body_type` 허용값: 경차·소형차·준중형차·중형차·대형차·스포츠카·SUV·RV·경승합차·승합차·화물차·기타.
> 정규화 단위: 주행거리 "만km" → ×10000, 가격 "천만원"=10,000,000 / "만원"=10,000.
