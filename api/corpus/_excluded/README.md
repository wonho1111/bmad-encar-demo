# 제외된 가이드 문서 (임베딩에서 빠짐)

이 폴더의 문서는 의도적으로 RAG 코퍼스에서 제외했다.

- `backfill_embeddings.py`의 `load_corpus()`는 `corpus/*.md`를 **비재귀**로 읽으므로,
  하위 폴더(`_excluded/`)에 둔 파일은 자동으로 `guide_documents` 적재 대상에서 빠진다.
- 파일은 삭제하지 않고 보존한다(되돌리기 쉽게).

## 왜 뺐나 (party-mode 토론 2026-06-23 결정)

이 서비스의 Job은 "매물 찾기"다. 경로 분류 원칙: **"이 답이 어떤 매물을 보여줄지 바꾸는가?"**
→ 바꾸면 B(문서 추천), 안 바꾸면 C(거절).

- `08-할부-리스-현금-비교.md`, `09-보험-세금-기초.md` 는 금융·세금 일반지식이라
  **어떤 매물을 보여줄지 바꾸지 않는다.** → 매물 추천 근거가 아니므로 라우터가 C(거절)로 보낸다.
- 코퍼스에 남겨 두면 라우터가 이 주제를 B로 흘릴 때 엉뚱한 근거 문서로 채택될 수 있어 제외했다.

되돌리려면 이 두 파일을 다시 `corpus/`로 옮기고 backfill을 재실행하면 된다.
