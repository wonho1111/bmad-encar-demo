# E2E 테스트 체크리스트 (웹 + 모바일)

엔카 데모(중고차 직거래)의 수동/자동 E2E 점검 목록. 웹은 Playwright 또는 브라우저 수동, 모바일은 실기기 또는 에뮬레이터로 1회씩 순회한다. 각 항목 `[ ]`에 결과 체크, 문제는 "비고"에 기록.

> 범례: ✅통과 / ❌실패 / ⚠️부분 / ➖해당없음

- **웹 실행**: `web/` 에 `.env.local`(NEXT_PUBLIC_SUPABASE_*·NEXT_PUBLIC_API_BASE_URL) → `npm run dev`(:3000) 또는 배포 프리뷰.
- **모바일 실행**: `app/` 에서 `flutter build apk --release --dart-define-from-file=.env.json` → 기기 설치(사이드로드). 시드 매물 100건·계정 위 표 기준.
- **선결 확인**: dev API 살아있음(`POST /ai/search` 200), Supabase RLS 정상.

## 1. 웹 (브라우저)

### 1.1 인증
- [ ] 비로그인으로 홈(`/`) → "로그인/회원가입" 노출, 보호 경로(`/search` 등) 접근 시 로그인으로 리다이렉트(307)
- [ ] 회원가입: 신규 이메일 + 역할(구매자/판매자) → 가입·자동 로그인 → 홈
- [ ] 회원가입: **이미 가입된 이메일** → "이미 가입된 이메일입니다" 에러
- [ ] 로그인: 올바른 계정 → 홈 / 틀린 비번 → 한국어 에러
- [ ] 로그아웃 → 비로그인 홈으로

### 1.2 매물 탐색·필터
- [ ] `/search` 진입 시 전체(판매중) 목록 표시(시드 100건)
- [ ] 필터: 제조사·차종·색상·연료·변속기 드롭다운, 가격/연식 범위, 모델 키워드 → 검색 결과 반영
- [ ] 가격 min>max 입력 시 swap 보정, 빈 필터=전체
- [ ] 결과 카드 클릭 → 상세 이동

### 1.3 매물 상세
- [ ] 15필드 + 옵션 + 설명 + 판매자 표시, "판매중" 배지
- [ ] 없는 id / sold / 삭제된 매물 → "찾을 수 없음" 안내(구매자 비노출)
- [ ] 본인 매물이면 "문의하기" 미노출 / 타인 매물이면 노출

### 1.4 AI 검색
- [ ] `/ai` 자연어 질의(예: "3천만원 이하 흰색 SUV") → 답변 + 매물 카드
- [ ] 멀티턴(이전 맥락 반영) 동작
- [ ] 빈 질의 차단, 너무 긴 질의 안내, 서버 오류 시 한국어 에러 + 입력 복원
- [ ] 결과 카드 클릭 → 상세

### 1.5 문의 채팅
- [ ] 상세 "문의하기" → 방 생성(중복 시 재사용) → 채팅방
- [ ] 메시지 전송 → 말풍선(본인 우측), 폴링으로 상대 메시지 수신
- [ ] 채팅 목록에서 방 재진입, 상대 표시(판매자/구매자)

### 1.6 판매(판매자 전용)
- [ ] 매물 등록: 15필드 입력 → 등록 → 즉시 노출(관리자 승인 없음)
- [ ] 내 매물 관리: 목록·상태 배지(판매중)
- [ ] 수정 → 반영 / 삭제 → 목록에서 제거
- [ ] 구매완료 → 확인 다이얼로그 → 판매중→판매완료, 구매자 탐색/상세에서 비노출

### 1.7 관리자(웹 전용)
- [ ] admin 로그인 → 관리자 화면 접근(회원/매물/거래/채팅 등 구현 범위)
- [ ] 구매자/판매자 계정으로 관리자 경로 접근 차단

### 1.8 권한·에러·빈 상태
- [ ] RLS: 타인 매물 수정/삭제 불가, 판매완료 매물 타인 비노출
- [ ] 네트워크/서버 오류 시 한국어 안내
- [ ] 빈 결과(검색 0건, 채팅 0개) 안내 문구

## 2. 모바일 (앱)

### 2.1 설치·기동
- [ ] **release APK 설치 후 정상 기동**(흰 화면/크래시 없음)
- [ ] **인터넷 동작**(로그인·조회 성공) — release `INTERNET` 권한 회귀
- [ ] 자동 로그인(앱 재실행 시 세션 유지), 환경변수 누락 시 안내 화면

### 2.2 인증
- [ ] 회원가입(역할 선택) → 자동 로그인 → 홈
- [ ] 중복 이메일 에러 / 틀린 비번 에러(한국어)
- [ ] 로그아웃 → 로그인 화면

### 2.3 홈 (신규 UI)
- [ ] AppBar "중고차 직거래" + **로그아웃 버튼이 또렷이 보임**(가시성 회귀)
- [ ] 프로필 카드: 역할 배지 + 이메일 표시
- [ ] 검색 CTA("어떤 차를 찾고 있나요?") → 매물 탐색
- [ ] 퀵액션: 문의 채팅(공통) / 매물 등록·내 매물 관리(판매자만 노출, 구매자 미노출)
- [ ] "최근 매물" 카드 노출(로딩/빈/에러 상태 포함), 카드 탭 → 상세, "더보기" → 탐색
- [ ] AI 검색 FAB → AI 화면

### 2.4 매물 탐색·상세
- [ ] 탐색 목록(시드 100건)·필터 동작, 카드 탭 → 상세
- [ ] 상세 15필드·옵션·설명·배지, 문의하기 노출 규칙

### 2.5 AI 검색
- [ ] 한글 질의 → 답변 + 매물 카드(dev API 실호출), 멀티턴, 에러 안내

### 2.6 문의 채팅
- [ ] 문의하기 → 방 생성 → 메시지 송수신·영속, 목록 재진입

### 2.7 판매(판매자)
- [ ] 매물 등록(드롭다운·한글 입력) → 성공 → 내 매물 관리 노출
- [ ] 수정·삭제·구매완료(다이얼로그 → 판매완료 전환)

### 2.8 관리자 차단
- [ ] admin 로그인 → "관리자 기능은 웹에서" 차단 화면 + 로그아웃만 가능(앱 기능 접근 불가)

### 2.9 모바일 고유
- [ ] **SafeArea/내비바**: AI 채팅·문의 채팅 입력란, 로그인·회원가입·매물등록 제출 버튼, 상세 "문의하기", 목록 마지막 항목이 **하단 시스템 바에 가리지 않음**(회귀)
- [ ] 키보드: 한글 입력 정상, 입력 시 폼 스크롤(가림 없음)
- [ ] 뒤로가기(시스템/AppBar ←) 동선 정상, 중복 화면 스택 없음
- [ ] 화면 회전/작은 화면에서 레이아웃 깨짐 없음(선택)
- [ ] 네트워크 끊김 시 한국어 오류 안내(흰 화면/무반응 금지)
